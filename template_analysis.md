# Template Analysis - SIEPE Product Data Sheet

## Original Template Layout (Deckelfässer_30_-_60_L_PSTG.pdf)

### Header
- Product name: Large, bold, uppercase, dark gray
- Subtitle: Below product name, uppercase, smaller

### Main Content Area
- **Left side (40-45%)**: Product image
- **Right side (55-60%)**: Description sections
  - MATERIAL
  - FARBE
  - AUSFÜHRUNG
  - VARIANTEN
  - PACKWEISE

### Technical Data Table
- Title: "TECHNISCHE DATEN" (uppercase, gray)
- Horizontal divider line above
- Table with:
  - First column: Property labels (left-aligned)
  - Data columns: Values (center-aligned)
  - Alternating row colors (white/light gray)
  - Column headers: 30 L, 35 L, 40 L, 50 L, 60 L

### Footer Section
- Horizontal divider line
- Footer note text (small, gray)
- Company info block:
  - Company name, website, email (left)
  - Multiple locations with vertical dividers between them
  - Each location: Name, Street, ZIP+City, Phone
- SIEPE logo (bottom right, large)
- Document number (bottom left, small gray)

## Key Differences to Fix in Generated PDF

1. Image positioning - should be on the LEFT side
2. Description sections - should be on the RIGHT side
3. Table column headers should show size values (30 L, 35 L, etc.)
4. Footer layout needs vertical dividers between locations
5. Logo should be larger and positioned at bottom right
